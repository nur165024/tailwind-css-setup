# tailwind css setup in your window pc

## first command line

```sh
  npm init -y
```

## second command line

```sh
  npm install -D tailwindcss postcss autoprefixer vite
```

or

```sh
  npm install -D tailwindcss
  npm install -D postcss
  npm install -D autoprefixer
  npm install -D vite
```

## third

```json
  "scripts": {
    "dev": "vite"
  }
```

## four command line

```sh
  npx tailwindcss init -p
```

## five command line

```sh
  npm run dev
```
